Detailed Description

Support for recognizing compression suffixes on data files. Compression of a file is recognized in mapreduce by looking for suffixes of supported codecs. For now Impala supports LZO, GZIP, SNAPPY, and BZIP2. LZO can use the specific HIVE input class.
