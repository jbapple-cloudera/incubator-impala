Detailed Description

Represents the result of an asynchronous Table loading request. Calling get() will block until the Table has completed loading. When finished processing the request, call close() to clean up.

void com.cloudera.impala.catalog.TableLoadingMgr.LoadRequest.close ( )
inline

Cleans up the in-flight load request matching the given table name. Will not cancel the load if it is still in progress, frees a slot should another load for the same table come in. Can be called multiple times.
