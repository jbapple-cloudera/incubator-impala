Detailed Description

Callers may pass a StatsMetric to gather the latency distribution of task execution.

Calls function(args[i]) num_args times in parallel using num_args threads. If any of the work item fails, returns the Status of the first failed work item. Otherwise, returns Status::OK when all work items have been executed.
