static Byte com.cloudera.impala.catalog.HdfsStorageDescriptor.parseDelim ( String  delimVal)
inlinestatic

Parses a delimiter in a similar way as Hive, with some additional error checking. A delimiter must fit in a single byte and can be specified in the following formats, as far as I can tell (there isn't documentation):

  • A single ASCII or unicode character (ex. '|')
  • An escape character in octal format (ex. \001. Stored in the metastore as a unicode character: ).
  • A signed decimal integer in the range [-128:127]. Used to support delimiters for ASCII character values between 128-255 (-2 maps to ASCII 254).

The delimiter is first parsed as a decimal number. If the parsing succeeds AND the resulting value fits in a signed byte, the byte value of the parsed int is returned. Otherwise, if the string has a single char, the byte value of this char is returned. If the delimiter is invalid, null will be returned.
